# Arduino Development Skill

## 简介
这是一个为 Arduino 全流程开发设计的 Skill，旨在将一句话需求自动转化为编译和烧录验证的完整开发流程。它基于 `arduino-cli` 工具，支持 AVR（如 Uno/Nano/Mega）、ESP32 系列、SG200X RISC-V 以及其他所有 Arduino 支持的开发板。

## 功能内容
- **项目创建与骨架生成**：针对不同硬件需求自动生成 `.ino` 文件，以及分离模块的 C++ 源码。
- **依赖自动解析**：自动分析代码中的 `#include`，并调用 `arduino-cli` 安装对应的第三方库。
- **编译与错误修复**：自动识别 FQBN，执行代码编译，并在发生警告或错误时进行智能解析与自我修复。
- **自动烧录**：自动识别连接到主机的 COM 端口并将固件上传。
- **串口调试与监控**：一键开启串口调试输出（支持后台运行机制）。

## 如何使用
将本文件夹部署到你的 Agent 插件或 Skill 目录中。在与 Agent 对话时，你可以直接提出需求：
- "帮我写一个ESP32连接WiFi并上报温湿度数据的程序，并烧录到板子上。"
- "创建一个 Arduino Uno 的项目，控制两个舵机。"
- "检查并修复我的 ESP32 编译错误。"

## 注意事项
1. **环境依赖**：系统必须预先安装 `arduino-cli`，并添加至环境变量 `PATH`。如果不具备，脚本会尝试提供初始化引导。
2. **引脚冲突限制**：在进行 ESP32 等设备的开发时，Claude 会自动检查引脚冲突（如避免使用 strapping 引脚或与 ADC2 冲突），**请务必确保实际连线与代码分配一致**。
3. **后台进程管理**：如果开启了串口监控（Monitor），它将作为一个长期后台任务运行，你需要使用任务管理相关的命令停止它。
